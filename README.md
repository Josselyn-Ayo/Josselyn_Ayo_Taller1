Calculadora en JavaFX
# DESCRIPCCIÓN
Este taller es una calculadora básica desarrollada en java utilizando JavaFX.
Permite realizar operaciones simples como la suma, resta, multiplicación y división.
Esta estructurado con el patrón Modelo, Controlador y Vista.
# Funcionalidades
 Suma de dos números
 Resta de dos números
 Multiplicación
 División (con validación de división por 0)
 Validación de datos (solo números)

 # Estructura del proyecto
Modelo (ModeloCalculadora.java)
Contiene la lógica de las operaciones matemáticas.
Controlador (ControladorCalculadora.java)
Gestiona la interacción entre la vista y el modelo.
Vista (Calculadora.fxml)
Interfaz gráfica construida con JavaFX.
Main (Main.java)
Punto de inicio de la aplicación.
# Notas importantes
La división entre 0 muestra un mensaje de error.
Si se ingresan letras u otros caracteres, se muestra una advertencia.
